## Overview

Pegasus-Suite is an all-in-one cybersecurity framework designed for security professionals, penetration testers, and ethical hackers. This toolkit provides a unified interface to access over 1000 specialized security tools for reconnaissance, vulnerability assessment, exploitation, post-exploitation, and more.

## Usage
1. Clone the repository
2. Run `python Pegasus-Suite.py`
3. Navigate through the interactive menu to access the desired tools

## Installation
```
git clone https://github.com/sobri3195/Pegasus-Suite.git
cd Pegasus-Suite
python Pegasus-Suite.py
```
